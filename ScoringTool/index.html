3) Build a 25–100 Mile Partner Radius List Source-based partner discovery — no invented descriptions
Partner radius workflow: First add the seed district, health center, college, or technical school sites above. Then choose an anchor site, select a radius, and search official K-12, HRSA healthcare, and higher-ed source data for nearby potential partners. Descriptions are assembled only from returned source fields; blank or unavailable facts stay blank.
High-fit means the provisional D-1 + D-2 objective score is 50+ out of 70, using available Census/SAIPE data. This is a screening flag, not a USDA decision.
Click a partner column heading to sort results. Distance, D-1, D-2, and Total score sort numerically.

Project Site / Source File Import

NCES import fix v18: K-12 district search now starts from the bundled ELSI/NCES 2025 district CSV with websites and # Schools, then loads school sites by LEAID, so it is less likely to fail on school-layer query parameters. # Schools is captured from the district record. You can now upload CSV/XLSX files from NCES/CCD school or district exports. Healthcare Lookup Dataset Fallback

Use this when HRSA live loading fails: Download the official HRSA Health Center Service Delivery and Look-Alike Sites XLSX, then upload it below. This loads the HRSA lookup dataset into the tool; it does not immediately score every site until you choose which results to add.
