Data + API Sources

This page documents the source tables, APIs, variables, and endpoint patterns used by the Scoring Tool, Prospect Finder, Prior Winners view, and Match Calculator. API examples omit the Census key value; the code appends the configured key at runtime.

FY2026 scoring source alignment

D-1 Rurality: uses the 2020 Decennial Census for Census place population and still requires USDA DLT map verification for rurality/non-rural/contiguity. D-2 Economic Need: uses county SAIPE “poverty percentage, all ages” from the 2023 SAIPE release published December 2024, matching the FY2026 guide’s referenced SAIPE file.

Notes for reps

Blank population or SAIPE values do not mean the site is ineligible. They mean the automated place/county match did not find a confident public API match. Use the USDA DLT map, Census Data Explorer, and county/FIPS verification before relying on the row for final scoring.

Static-site caveat

This GitHub Pages tool runs browser-side. Public datasets may fail because of CORS, rate limits, network settings, or source changes. Where that happens, use the import/fallback sections and keep the downloaded source file with the project record.
