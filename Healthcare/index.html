Data-source strategy

Primary list: HRSA Health Center Service Delivery and Look-Alike Sites

Best source for prospect lookup fields such as site name, organization, address, city, state, ZIP, phone, website, county, site type, and coordinates. This page should load it from data/hrsa_sites.csv.

Rural filters: USDA RUCA ZIP + HRSA FORHP rural ZIP approximation

RUCA supports rural-urban sorting/filtering. FORHP ZIP approximation gives a practical rural yes/no flag for rural health prospecting.

Enrichment: CMS FQHC Enrollments + CMS FQHC All Owners

CMS is useful for Medicare enrollment and ownership context. It should not replace HRSA as the site prospecting base.

Rep-owned fields

Direct contact emails are usually not in official source files. Reps should add verified Contact Name, Contact Email, Assigned To, Status, Priority, Follow-up Date, and Notes in the details panel, then export CSV.

This package includes a GitHub Action and Python script to refresh local files server-side. That avoids browser CORS blocks.
