# Source Strategy

## Best primary source

Use HRSA Health Center Service Delivery and Look-Alike Sites as the base FQHC/site prospecting list. It is designed for health center/service site lookup and includes site-level details such as name, address, phone, URL, site type, and location fields when available.

## CMS source use

CMS FQHC Enrollments is useful for Medicare enrollment validation and legal/DBA naming.

CMS FQHC All Owners is useful for ownership intelligence, not as the primary prospect list.

## RUCA and rural

RUCA is ZIP-level enrichment from USDA ERS. FORHP rural is a rural health grant eligibility approximation from HRSA. The tracker includes both because sales teams often need both a broad rural/likely rural flag and the RUCA code for filtering.

## Email limitations

Official datasets do not consistently publish prospecting emails for FQHC sites. The tracker provides email fields, but the system will only display emails if the official source includes them or if reps add verified contacts.
