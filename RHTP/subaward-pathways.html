Pathway definitions

Use these definitions to quickly understand what the state is asking for and where a sales rep can engage.

LOI Process

A brief concept submission used to align an idea with state priorities, screen eligibility, and determine whether a future readiness, planning, implementation, or targeted path fits.

Readiness Pathway

For organizations that need pre-project capacity building, eligibility cleanup, partner alignment, or basic operational readiness before a larger application.

Planning Pathway

For design, feasibility, needs assessment, stakeholder planning, budget development, or strategic planning before implementation.

Project Implementation Application

For ready-to-launch projects with defined scope, budget, timeline, outcomes, and implementation partners.

RFA / Grant Application

Competitive grant application route for eligible providers, CBOs, schools, coalitions, hospitals, or public entities.

RFGA / RFGP / NOFO

Grant-proposal variants used by states to solicit full project proposals; treat as grant applications unless the package says otherwise.

RFP / Procurement Contract

Vendor/contractor route for services such as TA, PMO, analytics, implementation, evaluation, infrastructure, or program administration.

RFI / Public Input

Early-stage information gathering; not usually an award route, but useful to shape state requirements and create sales visibility.

Invite-Only / Targeted Solicitation

Restricted route where the state identifies eligible organizations or model participants and invites LOIs/proposals.

Technical Assistance Application / TA Call

Application support or separate contractor/participant path for organizations needing help to prepare, implement, report, or comply.

Direct Allocation / Payment Certification

Nontraditional path where eligible entities submit certification, budget, or direct-allocation materials rather than a full competitive RFA.

Subrecipient Budget / Agreement

State-to-awardee pass-through structure; awarded applicants execute grant/subaward agreements and may budget partners/subrecipients when permitted.

Advisory / Stakeholder Pathway

A formal council, nomination, webinar, forum, or partner input route that does not itself award funds but may shape future awards.

Monitor / Not Yet Posted

No public application package verified; sales should watch the official state RHTP page and procurement/grants portal.
