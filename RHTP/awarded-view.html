Refresh rules for Awarded View

Use this checklist on every state refresh
1. Awarded trigger

Only mark Awarded when an official state source names awardees, selected leads, NOIA recipients, contract awards, subgrant recipients, or funded sites.

2. Do not count Y1 allocations

Federal CMS state allocation amounts are baseline state awards, not local subawards. They should not create an Awarded status by themselves.

3. Search terms

Check: awardees, awarded, selected, notice of intent to award, NOIA, recipients, grantees, contracts, subawards, subrecipients, hub leads.

4. Detail capture

Capture state, region, award category, recipient, purpose/equipment/role, county or region, award stage, source URL, and posted date when available.

5. Sales handoff

For named leads/awardees, create a partner/contact path and note whether Solutionz should pursue the awardee, a hub partner, or the next downstream applicant.

6. Quality control

Remove false positives where the word “award” only refers to the federal Y1 award amount, forecasted awards, or future expected awards.
