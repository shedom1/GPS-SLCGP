What's changed since the last update

  • Florida: moved to Closed. Florida's official SLCGP page and state one-pager now make clear that the posted application window ran October 15, 2025 to December 15, 2025, used the FDEM Grants Portal, and carried a 20% match. State page | One-pager
  • Pennsylvania: clarified How to Access Funding. The Commonwealth page now clearly says local governments will apply to the Commonwealth for committee-identified projects, cannot apply directly to the federal government, and should sign up for updates while additional application instructions are prepared. Source
  • West Virginia: clarified state messaging. WVEMD's homepage and cybersecurity funding page actively encourage eligible agencies to apply and provide a dedicated contact email, but the reviewed official pages still do not post a clearly dated state-local application packet or deadline. Homepage | Funding page
  • All other states: after a fresh 50-state check using current official state program pages and, where useful, homepage/news/announcement layers, I did not confirm another material status change that required a row change in the tracker.
