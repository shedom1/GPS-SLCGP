SLCGP Executive Summary of Changes

April 2026

April remains the highest-value month in the current cycle because the clearest live and near-term windows are clustered here. Immediate sales attention belongs on states with live deadlines, rolling participation, or a newly posted opening date.

Southeast

  • Georgia

    Georgia remains one of the clearest near-term opportunities on the board. The FY2024 window is still open through April 29, 2026, and the state has already signaled an FY2025 opening on September 1, 2026. Reps should treat today’s close as the immediate deadline and begin lining up FY2025 conversations now.

    Resource
  • South Carolina

    South Carolina remains in an active state-led application cycle. The SAA page says the FY2024 process began March 20, 2026 and applications are due by May 15, 2026, close of business. The updated worksheet also expands the objective set, which creates additional room for customer-aligned project discussions.

    Resource · Apply / details
  • Alabama

    Alabama continues to run an always-on, first-come participation model rather than a one-time local cash competition. The state page still shows four funded rounds and no direct cost to the applicant for approved services. This remains a practical sales conversation for local entities that want state-managed cybersecurity help without waiting for a narrow filing window.

    Resource
  • Florida

    Florida is not open today, but it stays on the watch list because the state page says the next FY24 cycle will be released later in 2026. No new dates are posted yet, so this is not a current filing opportunity. It is, however, a practical state for early customer education and list-building ahead of the next release.

    Resource

Mid-Atlantic

  • Maryland

    Maryland is still actionable because the state page keeps a rolling second-round FY2023 application open and separately flags a short-term project opportunity. The same page says FY2024 and FY2025 information is still forthcoming, so the immediate play is to work the currently open channels while monitoring for the next release. This is one of the cleaner rolling opportunities on the board.

    Resource

Northeast

  • Missouri

    Missouri has a clearly documented FY2025 competition open now through May 20, 2026 at 5:00 p.m. CST. The NOFO is explicit about the WebGrants process, the statewide planning expectations, and the $200,000 federal-share cap per applicant. This is one of the strongest current deadline-driven opportunities for near-term rep outreach.

    Resource · Apply / details

West

  • Washington

    Washington moves back into the actionable pipeline because the official WaTech page now advertises the next round opening May 4, 2026, with applications due June 12, 2026. The page also continues to reference the State Planning Committee and webinar resources, which gives reps a cleaner program trail than the tracker previously had. This is now a true upcoming-window state rather than a generic monitor-only entry.

    Resource
  • Wyoming

    Wyoming remains open and deadline-driven. The state homepage says SLCGP applications are being accepted now and must be submitted in ZoomGrants by May 15, 2026. This is one of the clearer western opportunities for immediate rep follow-up.

    Resource
  • Idaho

    Idaho remains active through a participation-request model rather than a traditional grant competition. The Operation Cyber Idaho page still invites cities, counties, school districts, tribes, and community partners to register for SLCGP-funded services. This is a strong fit for consultative outreach focused on service adoption and technical assistance.

    Resource
  • New Mexico

    New Mexico continues to accept participation requests for state-managed SLCGP services tied to State Fiscal Year 2027 delivery. The current public path is through the participation request form rather than a standard local cash-subaward window. Reps should position around shared services and program participation rather than a typical competitive application.

    Resource
  • Oklahoma

    Oklahoma still posts quarterly 2026 deadlines, including April 30, July 31, and October 30, but the state also says funding is not currently accessible and that additional funding is expected later this year. That makes this a monitor-and-engage state rather than a pure submit-now opportunity. The public page is still worth watching because the structure for future rounds is already in place.

    Resource
  • Oregon

    Oregon is not open today, but it remains important because the state page now clearly says the SLCGP funding opportunity is temporarily closed and expected to reopen in Summer 2026. This is a clean upcoming-watch state for pipeline planning. The immediate sales move is to identify likely prospects and be ready before the reopening notice posts.

    Resource

March 2026

March produced fewer broad state-window changes, but it still included specific, dated guidance in a small number of states. Closed rounds from this month remain useful benchmarks for how states are structuring focused project asks.

Northeast

  • Ohio

    Ohio’s Round 3 .gov migration guidance was one of the more specific March actions in the tracker. The state required applications by March 13, 2026 and capped federal support at $9,500 per project with a 20% cost share. That round is closed, but it remains a good reference point for the kind of narrowly defined project structures states may continue to use.

    Resource

February 2026

February was more operational than expansionary. The strongest signal in this month was on program continuity, implementation, and the impact of federal funding disruptions.

Northeast

  • Maine

    Maine remains a watch state, not an active filing state. Its page says the Year 1 and Year 2 participation windows are closed, projects are underway, and a partial federal shutdown has temporarily paused a funded KnowBe4 training service. For leadership, the main takeaway is that the program is still live in operations, but not currently open for a new broad participation round.

    Resource

January 2026

January established several of the first clearly dated 2026 state actions. Most of these windows are now closed, but they remain useful signals for state cadence, priorities, and planning-committee involvement.

Southeast

  • Louisiana

    Louisiana’s January cycle covered FY23 through FY25 activity and ran from January 1 to February 28, 2026. State materials emphasized mitigation and training, reinforcing the state’s preference for practical implementation use cases. The round is closed, but the current record gives reps a useful read on what Louisiana appears to value.

    Resource
  • Mississippi

    Mississippi’s 2026 application opened January 15 and closed February 27, 2026, with public grant-writing sessions held in late January and early February. The state also says Batch 4 participation has not yet been determined. That means the current round is closed, but the page is still worth watching for the next batch decision.

    Resource

Northeast

  • Kansas

    Kansas opened a clearly dated January 1 to February 28, 2026 submission window and now says Phase 2 applications are under State Cybersecurity Planning Committee review before federal review and approval. The active submission period is over, but the committee process remains visible. This keeps Kansas relevant as a monitoring state for award and follow-on updates.

    Resource

West

  • California

    California’s key January action was a formal FFY2024 local and tribal competition with a March 13, 2026 due date. That round is closed, but it remains one of the clearest examples of a formal state RFP-style structure. It is best viewed now as a completed benchmark rather than a current rep target.
