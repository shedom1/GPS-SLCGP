# RUS DLT GitHub Static Dashboard

This package is a read-only GitHub Pages version.

What it does:
- pulls **K-12** rows live from the public Google Sheet
- keeps **Tier 1** and **Tier 2** using the included tier map
- includes **Tier 3** and **Tier 4** higher-ed rows from the local JSON files in `data/`
- shows a detail card when you click an organization name

## Publish on GitHub Pages
1. Unzip this package.
2. Upload the contents into your `/RUS-DLT/` folder in GitHub.
3. Make sure `index.html` sits at the root of that folder.
4. Keep the `data/` folder next to it.
5. Commit the files.

## Notes
- This version is intentionally **read-only**.
- HubSpot remains the official place for sales activity.
- If the K-12 sheet name changes from `ELSI Export`, the page will need a quick update.
