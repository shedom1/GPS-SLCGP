This page is set up to work with the shared Google Sheets login log.

Login Log

Use the Google Sheet as the shared audit log for site activity. The homepage writes login and logout events to that sheet through the Google Apps Script web app.

Open Google Sheet Back to Homepage

Update the GOOGLE_SHEET_URL constant in this file after you create the sheet.

What this setup records

  • Name entered at login
  • Login or logout action
  • Reason, including 24-hour session expiration
  • Timestamp
  • Page URL
  • Browser user agent

Deployment reminder

Keep this page discreet at the bottom of the homepage, and use the Apps Script deployment URL in the homepage file so all users write to the same shared sheet.
